# tic-tac-toe
# ❌⭕ Tic Tac Toe Game

This project is a web-based implementation of the classic Tic Tac Toe game developed using HTML, CSS, and JavaScript. It provides an interactive and engaging user experience where two players can play against each other in a simple and responsive interface.

The project focuses on implementing core front-end development concepts such as DOM manipulation, event handling, and game logic using JavaScript.

---

## 🚀 Features

- ❌⭕ Two-player gameplay (Player X and Player O)
- 🔄 Turn-based system with automatic switching
- 🏆 Winner detection based on all possible winning combinations
- 🤝 Draw detection when no moves are left
- 🔁 Restart / New Game functionality
- 📱 Fully responsive design (works on mobile, tablet, desktop)
- 🎨 Clean and modern user interface

---

## 🛠️ Tech Stack

- **HTML5** – Structure of the game board
- **CSS3** – Styling, layout, and responsiveness
- **JavaScript (ES6)** – Game logic, DOM manipulation, and interactivity

## ⚙️ How It Works

- The game consists of a 3x3 grid.
- Two players take turns placing their marks (X or O).
- JavaScript tracks each move and updates the UI dynamically.
- After every move:
  - The system checks for a winning condition.
  - If a player wins, a message is displayed.
  - If all cells are filled without a winner, the game ends in a draw.
- Users can restart the game anytime using the reset button.
